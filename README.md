# tao-web
Angular web app for tao-wallet.

### install

Clone the repo and navigate to it:
```
git clone https://github.com/cold-sats/tao-web
cd tao-web
```

Run npm install:
```npm install
```

Install Angular CLI globally:
```
npm install -g @angular/cli
```

Run on http://localhost:4200/#/:
```
ng serve --open
```
